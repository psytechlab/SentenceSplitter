# Анализ ошибок Opus 4.8 — 20 задач с худшим boundary F1 relaxed

Метрика: avg boundary F1 relaxed (±3) Opus против Даниила, Ширин и Игоря.  
Сортировка по возрастанию F1.

> **Контекст:** у Opus худшие 20 задач имеют F1 от 0.000 до 0.667 — то есть даже «худшие» случаи в основном это **расхождение в точке разреза при согласии по типу**, а не грубые ошибки классификации. Для сравнения: у Sonnet 7 задач имели F1=0.000.

## Подробный разбор

### [156] F1 = 0.000 — Категория A (единственная содержательная ошибка типа)
**Текст:** «Я стала вновь жить, радоваться, ... поступила туда, куда хотела, ходила в церковь.»  
**Opus:** СПП, 3 спана («Я стала... поступила туда» / «куда хотела» / «ходила в церковь»). **Люди:** Не сложное, 1 спан.  
**Причина:** Opus применил правило придаточной места «куда хотела» (восстановимый предикат «поступить» → отдельная клауза, 2.4.2). Люди трактовали «куда хотела» как двойной эллипсис (чеклист п.6) и не разделяли. Это **легитимный пограничный случай** — чеклист сам себе противоречит (пример в п.6 говорит «ДА, разделять»). Opus следует букве правила СПП, люди — интуиции «это один рассказ».

### [130] F1 = 0.222 — Категория C
**Текст:** «Что странно вернул меня и стал совсем другим человеком: ласковый...»  
Opus: «Что странно» / «вернул и стал» (однород.) / «ласковый...». Daniil: «Что странно» / «вернул меня и стал...». Расхождение — куда отнести двоеточие-перечисление «ласковый, романтичный, нежный». Тип (Смешанного) совпал.

### [1] F1 = 0.333 — Категория F
«/ Я фанат странных незнакомцев ))))))» — Opus и все люди дали 1 спан «Не сложное». F1 занижен из-за трейлинговых `))))))`, которые нормализатор по-разному обрезает.

### [2], [63] F1 = 0.333 — Категория F
СПП, 2 спана, тип совпал с людьми. Off-by-one в позиции границы (конечный знак / 1–2 символа).

### [124] F1 = 0.333 — Категория C
«Украинский народ, помня, какие жертвы...» Opus: «Украинский народ» / «помня, какие жертвы принесла» / «никогда не простит и будет мстить». Daniil режет деепричастный блок иначе («Украинский народ, помня,» / «какие жертвы...»). Тип СПП совпал; разница — где проходит граница внутри деепричастного оборота с изъяснительной.

### [148] F1 = 0.333 — Категория F/C
«Я начинающая певица — певица позвольте заметить...» Opus и Daniil: 1 спан «Не сложное». Shirin разбил (БСП). F1 занижен мажоритарным усреднением по трём людям, двое из которых дали другую зернистость.

### [116] F1 = 0.500 — Категория C
«У меня все хорошо, трое замечательных детей, муж, мама.» Opus: 2 спана БСП (совпало с людьми по типу и структуре). **Это улучшение над Sonnet**, который дал 1 спан «Не сложное».

### [142], [103] F1 ≈ 0.51–0.55 — Категория D
Очень длинные тексты потока сознания (300–400 симв.). Opus дал 5 и 6 спанов, Daniil — 8 и 15. Недоразбиение длинных цепочек — общая для обеих моделей слабость.

### [36], [151] F1 = 0.600 — Категория E
Вставная «как мне кажется» (36) и цель «чтобы стать первым» (151): Opus выделяет вставную/придаточную более дробно, чем Daniil, который объединяет. Тип у Opus (Смешанного) формально точнее, но не совпал с Daniil (СПП) — расхождение в трактовке вставной.

### [168] F1 = 0.633, [79] F1 = 0.643 — Категория C
Разная зернистость БСП/СПП-цепочек: люди дробят на 1–2 спана больше, тип совпал.

### [119] F1 = 0.533 — Категория C
«У меня мягкий характер, я весёлая...» Opus: 4 спана, тип Смешанного; Daniil: 3 спана, БСП. **Улучшение над Sonnet** (который дал 2 спана ССП с F1=0.000).

### [21], [22], [44] F1 = 0.667 — Категория F
Все три — «Не сложное», 1 спан, полное согласие с людьми по типу и структуре. F1 занижен только из-за того, что у некоторых людей внутри есть лишняя «техническая» граница (тире-приложение), которую нормализатор считает.

**Главное:** из 20 худших задач только **4** имеют расхождение по типу с majority (156, 119, 36, 151), и все четыре — пограничные случаи (двойной эллипсис, БСП↔Смешанного, трактовка вставной). **Для доработки промпта:**
1. Двойной эллипсис «куда хотела» (156) — устранить противоречие в чеклисте п.6: явно решить, разделять или нет.
2. Длинные тексты >300 символов (103, 142) — инструкция дробить мельче.
3. Вставные/придаточные (36, 151) — калибровать зернистость под человеческую (люди чаще объединяют определительную со вставной).
