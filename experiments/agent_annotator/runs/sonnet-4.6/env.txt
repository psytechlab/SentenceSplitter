Воспроизведение:
  1. Склонировать репозиторий.
  2. Установить зависимости: pip install -r requirements.txt (или аналог).
  3. Восстановить inputs/*_input.json из validation/ground_truth.csv:
       python3 -c "
       import csv,json
       with open('validation/ground_truth.csv') as f:
           gt=sorted(csv.DictReader(f), key=lambda r: int(r['task_id']))
       for b in range(1,11):
           items=[{'task_id':int(r['task_id']),'text':r['text']} for r in gt[(b-1)*20:b*20]]
           with open(f'experiments/agent_annotator/inputs/batch_{b:02d}_input.json','w') as f:
               json.dump(items,f,ensure_ascii=False,indent=2)
       "
  4. Запустить Claude Code с системным промптом = prompt/instructions_snapshot.md + prompt/few_shot_examples.md + prompt/checklist.md.
  5. Для каждого inputs/batch_NN_input.json: попросить аннотировать 20 текстов, вернуть JSON-массив → runs/<модель>/outputs/batch_NN.json.
  6. Склеить в runs/<модель>/annotations.json.
  7. Запустить metrics: python3 validation/lib.py (или скрипт из README.md).

Примечание: прогон выполнен Claude через Claude Code batch-аннотацию (не через API напрямую).
