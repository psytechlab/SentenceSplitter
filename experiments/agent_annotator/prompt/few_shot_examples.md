# Few-shot примеры — 6 эталонных разметок

Все 6 примеров взяты из validation/ground_truth.csv — задачи, где **все три аннотатора согласились единогласно** (daniil_type == shirin_type == igor_type).  
Spans указаны в символьных смещениях (start:end, exclusive), строго соответствуют разметке аннотаторов.

---

## Пример 1 — «Не сложное»

**task_id:** 5  
**stratum:** solyanka_lenta_long

**Текст (122 символа):**
```
«Речь идет о большой группе злоумышленников, совершивших несколько заказных убийств и покушений на убийств в городе Киеве.
```

**Разметка:**
```json
{
  "task_id": 5,
  "spans": [
    {"start": 0, "end": 122, "text": "«Речь идет о большой группе злоумышленников, совершивших несколько заказных убийств и покушений на убийств в городе Киеве."}
  ],
  "sentence_type": "Не сложное",
  "reasoning": "Одна грамматическая основа: «речь идёт». Оборот «совершивших несколько заказных убийств…» — причастный, не имеет собственного сказуемого (правило 2.6.2), не разделяем."
}
```

## Пример 2 — «ССП»

**task_id:** 157  
**stratum:** presuicidal_multilabel_conj

**Текст (56 символов):**
```
Я схожу с ума и это тоже заставляет меня сильно грустить
```

**Разметка:**
```json
{
  "task_id": 157,
  "spans": [
    {"start": 0, "end": 13, "text": "Я схожу с ума"},
    {"start": 14, "end": 56, "text": "и это тоже заставляет меня сильно грустить"}
  ],
  "sentence_type": "ССП",
  "reasoning": "Два предикативных блока с разными основами (я схожу; это заставляет), соединены сочинительным союзом «и» (правило 2.3.1). Граница перед союзом «и»."
}
```

## Пример 3 — «СПП» с разорванным главным (интерполированное придаточное)

**task_id:** 17  
**stratum:** solyanka_lenta_long

**Текст (174 символа):**
```
Весь год, пока ресторан был закрыт, он вместе с помощниками ездил по скандинавским странам и искал редкие традиционные блюда: «от странных растений и ракушек до рыбьих глаз».
```

**Разметка:**
```json
{
  "task_id": 17,
  "spans": [
    {"start": 0, "end": 8, "text": "Весь год"},
    {"start": 10, "end": 34, "text": "пока ресторан был закрыт"},
    {"start": 36, "end": 174, "text": "он вместе с помощниками ездил по скандинавским странам и искал редкие традиционные блюда: «от странных растений и ракушек до рыбьих глаз»."}
  ],
  "sentence_type": "СПП",
  "reasoning": "Главное предложение 'Весь год... он ездил по странам и искал блюда' разорвано встроенным придаточным времени 'пока ресторан был закрыт'. Физически это даёт 3 спана: 'Весь год' (начало главного) + 'пока ресторан был закрыт' (придаточное времени, 2.3.2) + 'он ездил...искал блюда' (продолжение главного). Спаны 1 и 3 — одна логическая клауза, разорванная встроенным придаточным; 'Весь год' — обстоятельство главной части, а не отдельная клауза. Однородные сказуемые 'ездил и искал' в третьем спане — одна клауза с одним подлежащим 'он' (правило 2.6.1, не разделять)."
}
```

## Пример 4 — «БСП» (два разных подлежащих, перечисление)

**task_id:** 20  
**stratum:** presuicidal_multilabel_bsp

**Текст (74 символа):**
```
Всё еще усугубляют некоторые проблемы со здоровьем, я заикающийся очкарик.
```

**Разметка:**
```json
{
  "task_id": 20,
  "spans": [
    {"start": 0, "end": 50, "text": "Всё еще усугубляют некоторые проблемы со здоровьем"},
    {"start": 52, "end": 74, "text": "я заикающийся очкарик."}
  ],
  "sentence_type": "БСП",
  "reasoning": "Два самостоятельных предикативных блока с разными подлежащими ('проблемы усугубляют'; 'я — очкарик'), связанных только интонацией перечисления без союза (правило 2.3.3). Это каноничный БСП: запятая разделяет клаузы с разными основами."
}

## Пример 5 — «Смешанного типа»

**task_id:** 6  
**stratum:** presuicidal_monolabel_long

**Текст (142 символа):**
```
А еще на фото моего класса на пиксельный фотик Мне сегодня 17, и я этого совсем не ощущаю, это странно чуть-чуть Но в целом день был чудесным!
```

**Разметка:**
```json
{
  "task_id": 6,
  "spans": [
    {"start": 0, "end": 47, "text": "А еще на фото моего класса на пиксельный фотик "},
    {"start": 47, "end": 61, "text": "Мне сегодня 17"},
    {"start": 63, "end": 89, "text": "и я этого совсем не ощущаю"},
    {"start": 91, "end": 112, "text": "это странно чуть-чуть"},
    {"start": 113, "end": 142, "text": "Но в целом день был чудесным!"}
  ],
  "sentence_type": "Смешанного типа",
  "reasoning": "Пять самостоятельных клауз, смешанная связь: БСП между 1-2 (разрывов нет, но разные основы), ССП союз «и» перед 3-й, БСП перечисление между 3-4, ССП «Но» перед 5-й. Тип «Смешанного типа» потому что использованы одновременно и сочинительная связь, и бессоюзная."
}
```

## Пример 6 — Короткое «БСП» из социальной сети (без знаков препинания между клаузами)

**task_id:** 136  
**stratum:** presuicidal_short_simple

**Текст (50 символов):**
```
Я боюсь доверять людям мне недавно исполнилось 17!
```

**Разметка:**
```json
{
  "task_id": 136,
  "spans": [
    {"start": 0, "end": 22, "text": "Я боюсь доверять людям"},
    {"start": 23, "end": 50, "text": "мне недавно исполнилось 17!"}
  ],
  "sentence_type": "БСП",
  "reasoning": "Два предикативных блока с разными подлежащими (я; 17 лет — безличное), связанных только интонационно, без союза и без знаков препинания — типичный стиль социальных сетей. Граница проводится по смене основы (2.3.3, перечисление)."
}
```
