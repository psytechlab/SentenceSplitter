# Чеклист спорных случаев

Сверяйся с этим списком при каждом тексте перед финальным решением.

---

## Что НЕ разделять

**1. Однородные сказуемые при одном подлежащем → НЕ разделять (правило 2.6.1)**  
   *Пример:* «Она встала и пошла к окну.» — одна основа «она встала/пошла» → 1 спан.  
   *Образец ошибки:* «Тело онемело, затряслось, пошли слёзы, я задыхался.» (task_id=114) — «тело онемело, затряслось» это **однородные сказуемые при одном подлежащем** → одним спаном [0:24], НЕ двумя. Далее «пошли слёзы» и «я задыхался» — уже другие подлежащие → отдельные спаны. Итого 3 спана, не 4.  
   *Тест:* Есть ли у каждого глагола своё подлежащее? Если нет → один спан.

**2. Зависимый инфинитив при модальном глаголе → НЕ разделять**  
   *Пример:* «Я хочу уехать.» — «уехать» зависит от «хочу», одна основа → 1 спан.  
   *Тест:* Инфинитив отвечает на «что делать?» при модальном/фазовом глаголе → не клауза.

**3. Причастный оборот → НЕ разделять (правило 2.6.2)**  
   *Пример:* «Девочка, танцевавшая на сцене, упала.» — причастный оборот без сказуемого → 1 спан.  
   *Тест:* Есть ли у оборота собственное спрягаемое сказуемое? Если нет → не разделять.

**4. Деепричастный оборот → НЕ разделять (правило 2.6.2)**  
   *Пример:* «Уставший от дороги, он лёг спать.» → 1 спан.  
   *Тест:* Деепричастие ≠ самостоятельная клауза.

**5. Сравнительный оборот без сказуемого → НЕ разделять (правило 2.6.3)**  
   *Пример:* «Она бежала быстро, как ветер.» — «как ветер» без глагола → 1 спан.  
   *Тест:* Если после «как/словно/будто» нет сказуемого → не разделять.

**6. Двойной эллипсис типа «куда хотела» → НЕ отдельная клауза**  
   *Пример:* «Поступила туда, куда хотела [поступить].» — «куда хотела» это придаточная с восстановимым глаголом → ДА, разделять (2 основы). НО «куда хотела» без восстановимого подлежащего — проверить по контексту.  
   *Тест:* Можно ли восстановить полноценную основу? Если да → разделять.

**7. Вводные конструкции → НЕ разделять (правило 2.6.4)**  
   *Пример:* «К счастью, мы успели.», «Конечно, это трудно.» — вводные слова без основы → 1 спан.  
   *Тест:* Можно ли убрать конструкцию без потери грамматики? Если да → вводная → не разделять.

**8. Прямая речь в кавычках без самостоятельной реплики → НЕ разделять внутри (правило 2.6.5)**  
   *Пример:* «Она сказала: "Иди домой".» — прямая речь как объект при глаголе речи → 1 спан на всё.  
   *Тест:* Если прямая речь оформлена как вставка (кавычки), не выделяй её отдельно.

## Что разделять (осторожные случаи)

**9. Однородные придаточные с союзом «и» → РАЗДЕЛЯТЬ (правило 2.4.1)**  
   *Пример:* «Я не знаю, где он был и что делал.» → 3 спана: «Я не знаю», «где он был», «что делал».  
   *Тест:* Два придаточных соединены «и» → каждое выделяется отдельно, даже если кажутся одним целым.  
   ⚠ **Исключение:** Если после «и» стоит явный новый субъект (не тот, что у предыдущей придаточной), это ССП-клауза, а не вторая придаточная.  
   *Пример:* «Рассказали, что поступило заявление, **и теперь сотрудники** устанавливают личность.» → «сотрудники» = новый субъект при глаголе речи → «и теперь сотрудники устанавливают» = самостоятельная ССП-клауза → тип Смешанного (СПП + ССП).

**10. Указательные слова (туда, тогда, тот, там) → остаются в ГЛАВНОЙ части (правило 2.4.2)**  
    *Пример:* «Он поехал в тот дом, где жил раньше.» → «Он поехал в тот дом» (главная) + «где жил раньше» (придаточная). Указательное «тот» остаётся в главной.

**11. Вставные конструкции с собственным глаголом → РАЗДЕЛЯТЬ**  
    *Пример:* «Она — я это видел сам — солгала.» — «я это видел сам» имеет собственную основу → выделять как отдельный спан.  
    *Тест:* Есть ли у вставки своя грамматическая основа? Если да → разделять.

**12. БСП без знаков препинания (соцсети) → РАЗДЕЛЯТЬ если есть отдельные основы**  
    *Пример:* «Я боюсь доверять людям мне недавно исполнилось 17!» — две основы без запятой → 2 спана.  
    *Тест:* Найди основы. Если 2+ независимых основы без союза → БСП → разделять по смене основы.

## Типы предложений — краткий справочник

| Тип | Критерий |
|---|---|
| Не сложное | Ровно одна грамматическая основа (или однородные предикаты при одном подлежащем) |
| ССП | Две и более равноправных части, связаны сочинительным союзом (и, но, а, или, либо, зато...) |
| СПП | Главная + одна/несколько придаточных, связаны подчинительным союзом или союзным словом |
| БСП | Две и более частей с самостоятельными основами, без союзов, только интонация/знаки |
| Смешанного типа | В одном предложении используются 2+ разных типа связи |
